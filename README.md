# DSA Prep - TypeScript

Personal DSA practice repository.

## Tech Stack

- **TypeScript** for solutions
- **tsx** for running individual problem files
- **Jest** + **ts-jest** for unit tests
- **ESLint** + **Prettier** for code hygiene

## Commands

### Run a single problem (manual sanity check)

```bash
npm run dev src/patterns/fast-slow-pointer/0141-linked-list-cycle.ts
```

### Run a single problem (watch-mode)

```bash
npm run dev:watch src/patterns/fast-slow-pointer/0141-linked-list-cycle.ts
```

### Run all tests

```bash
npm test
```

### Run test for single problem

```bash
npx jest 0141
```

### Run test (watch-mode)

```bash
npm test:watch
```
